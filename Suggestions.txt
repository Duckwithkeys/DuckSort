DUCKSORT FUTURE ARCHITECTURAL & FEATURE SUGGESTIONS

1. ON-DEVICE AI & VISION MACHINE LEARNING
- Integrate Vision Framework (VNClassifyImageRequest and VNHumanBodyPoseRequest) to automatically auto-tag photo subjects, landscapes, and pets locally off the main thread.
- Implement Apple Neural Engine (ANE) acceleration for automated face clustering and duplicate detection without uploading private photos to third-party APIs.



2. PERCEPTUAL HASHING & BURST SHOT DEDUPLICATION
- Implement a perceptual hashing engine (pHash / dHash) to compute hamming distances between images, grouping burst shots and near-duplicate photos automatically.
- Provide automated "Best Shot" recommendations based on sharpness metrics (laplacian variance), exposure balance, and eye-open detection.



3. NON-DESTRUCTIVE BATCH EDITING PIPELINES
- Implement a non-destructive image processing pipeline using Core Image filters and metal-backed rendering contexts.
- Support batch applying color LUTs, white balance adjustments, and exposure corrections alongside XMP sidecar synchronization.



4. CLOUDKIT SYNC & CROSS-PLATFORM STATE HANDOFF
- Define a CloudKit sync engine for TagPacks, user preferences, and metadata edits so users can seamlessly sort libraries across multiple Macs or iPadOS devices.
- Support real-time conflict resolution when metadata sidecars are modified on external drives or networked storage (NAS).



5. PRO KEYBOARD WORKFLOW & ACCESSIBILITY
- Implement dedicated macOS menu commands and configurable global hotkeys for lightning-fast rating (1-5), flagging (Pick/Reject), and TagPack switching.
- Ensure full VoiceOver accessibility compliance with dynamic accessibility labels on thumbnail grids and metadata inspectors.
