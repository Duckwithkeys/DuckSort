================================================================================
  DUCKSORT — FEATURE IDEAS (50+)
  Based on full codebase analysis
================================================================================

These ideas are grouped by theme. Each entry includes a brief description
of what it would do and why it would be valuable to photographers.

================================================================================
  CULLING & REVIEW
================================================================================

1.  COMPARE MODE
    Side-by-side viewer for 2-4 photos simultaneously. Select multiple
    images and open them in a split-view to compare sharpness, exposure,
    and composition before picking the keeper. Essential for burst shots.

2.  FOCUS PEAKING OVERLAY
    In the large image viewer, overlay colored highlights on the sharpest
    edges in the photo to instantly assess focus accuracy without zooming.
    Toggle with a hotkey (e.g., "F").

3.  CLIPPING WARNING OVERLAY
    Highlight blown highlights in red and crushed shadows in blue over the
    large image preview. Helps photographers quickly identify exposure
    problems during culling.

4.  LOUPE VIEW (ZOOM WINDOW)
    A floating magnifier window that follows the mouse cursor and shows
    a 1:1 or 2:1 crop of wherever you hover. Useful for checking sharpness
    on eyes without needing to zoom the full image.

5.  FILMSTRIP NAVIGATION MODE
    A full-width filmstrip at the bottom with the large viewer above —
    like Lightroom's Loupe mode. Drag the filmstrip to scrub through photos
    quickly. Currently there's a FilmstripView but it could be the primary
    layout mode.

6.  AUTO-ADVANCE AFTER TAGGING
    Option to automatically move to the next photo after applying a tag
    or rating. Saves one keypress per photo in large sessions. Configurable
    delay (instant, 0.3s, 0.5s).

7.  REJECT & HIDE FLOW
    One-key action (e.g., "X" is already pick=-1) that rejects AND hides
    the photo from the current view, advancing automatically. Makes rapid
    culling feel like flipping through physical cards.

8.  STAR RATING HISTORY / UNDO
    Full undo/redo stack for rating, flag, and tag operations. Press Cmd+Z
    to undo the last tag or rating change, even across multiple photos.

9.  SMART SELECTION BY SIMILARITY
    Detect near-duplicate or burst photos (same base name or timestamp
    within N seconds) and auto-group them for quick bulk rejection.

10. FACE DETECTION HINT
    Use Vision framework to detect faces in thumbnails and show a small
    face count badge. Helps quickly identify "people" shots vs. landscapes.

================================================================================
  ORGANIZATION & WORKFLOW
================================================================================

11. COLLECTIONS / VIRTUAL ALBUMS
    Create named collections (like Lightroom Collections) that group photo
    sets without moving files. A photo can belong to multiple collections.
    Persisted to the tag store JSON.

12. SESSION HISTORY
    Track the last 10 scan sessions with timestamps, photo counts, and
    which folders were processed. One-click to re-open a previous session.

13. SMART FILTERS (SAVED FILTER PRESETS)
    Save the current combination of filter rule + tag filters + flag filters
    + rating filters as a named Smart Filter. Reapply with one click from
    the sidebar.

14. BATCH RENAME
    Rename exported files using a template (e.g., {EventName}_{Date}_{###})
    during copy/move operations. Preview the renamed filenames before
    confirming.

15. FOLDER WATCH / LIVE IMPORT
    Watch a source folder for new files (e.g., a tethered camera shooting
    directly to a folder). Auto-add new photo sets as they appear without
    requiring a manual rescan.

16. MULTI-DESTINATION EXPORT
    Export selected photos to more than one destination folder at once.
    Useful for backing up to an external drive and a cloud sync folder
    simultaneously.

17. EXPORT PRESETS
    Save full export configurations (destination, routing rule, operation
    type, JPEG only toggle) as named presets. One-click to run a saved
    workflow.

18. ARCHIVE MODE
    A special move operation that also compresses the source folder into
    a zip/archive after moving files out. Useful for archiving a full
    shoot after culling.

19. SUBFOLDER CREATION ON EXPORT
    During export, automatically create date-stamped subfolders (e.g.,
    2026-06-23/) in the destination even without a routing rule. Simple
    date-based organization without needing rule configuration.

20. IMPORT FROM CAMERA / MEMORY CARD
    Auto-detect when a camera or SD card is inserted (via NSWorkspace
    volume mount notifications) and offer to import immediately.

================================================================================
  METADATA & TAGGING
================================================================================

21. EXIF DISPLAY PANEL EXPANSION
    Show full EXIF data in the inspector: GPS coordinates, white balance,
    flash status, focal length, image dimensions, color profile, etc.

22. GPS MAP VIEW
    If photos have GPS coordinates, show a map (via MapKit) pinning the
    shoot location. Click a pin to filter to photos from that location.

23. CAPTION / DESCRIPTION FIELD
    Add a text field in the inspector to write a caption or description.
    Write it to the XMP sidecar as dc:description. Searchable.

24. COPYRIGHT / IPTC METADATA EMBEDDING
    Embed photographer name, copyright notice, and contact info into
    export sidecar files. Configure once in preferences, apply to all
    exports automatically.

25. AUTO-TAGGING BY EXIF
    Suggest tags based on EXIF data. If camera is a Fujifilm, auto-tag
    "Fuji". If focal length < 35mm, suggest "Wide Angle". User confirms
    or dismisses suggestions.

26. TAG HIERARCHY / SUB-TAGS
    Allow tags to have parent-child relationships. E.g., "Portrait" →
    "Environmental Portrait" → "Studio Portrait". Filter by parent selects
    all children.

27. TAG MERGE
    Combine two tags into one across all assignments. Useful when you
    realize "Graduate" and "Grad" should be the same tag.

28. TAG USAGE STATISTICS
    Show a stats panel: which tags are used most, which photos have the
    most tags, tag co-occurrence matrix (e.g., "Graduate" and "Family"
    always appear together).

29. COLOR LABEL SYSTEM
    Add a Finder-style color label to each photo set (red, orange, yellow,
    green, blue, purple, grey). Write to XMP as xmp:Label. Filter by color
    label in the sidebar.

30. KEYWORD HIERARCHY IMPORT
    Import a keyword hierarchy from a .txt or .lr-keyword file (Lightroom
    format) to bootstrap the tag categories and tags without manual entry.

================================================================================
  EXPORT & OUTPUT
================================================================================

31. JPEG CONVERSION ON EXPORT
    When exporting RAW files, offer to convert them to JPEG using ImageIO.
    Configure quality (0-100%) and color space. Write converted files to
    destination instead of (or alongside) the RAW originals.

32. WATERMARKING
    Stamp a text or image watermark onto exported JPEGs. Configure font,
    size, position, opacity. Applied via CoreImage during export.

33. CONTACT SHEET GENERATOR
    Generate a single PDF or JPEG contact sheet from selected photo sets,
    laid out in a configurable grid (e.g., 4×5 thumbnails with filenames).
    Useful for client proofing.

34. SHARING SHEET
    Share selected photos via macOS Share Sheet directly to Mail, AirDrop,
    Messages, or social apps. No export step needed for quick sharing.

35. SIDECAR XMP VALIDATION
    Scan all existing XMP sidecars and report broken, empty, or malformed
    files. Option to repair or regenerate them.

36. EXPORT LOG / REPORT
    After each transfer, generate a plain-text or CSV log of every file
    moved/copied with source path, destination path, timestamp, and success
    status.

37. DUPLICATE DETECTION ON EXPORT
    Before writing files to the destination, check for exact duplicates
    (by file hash) and skip them with a warning rather than creating
    conflicting copies.

================================================================================
  UI & USER EXPERIENCE
================================================================================

38. DARK / LIGHT / AUTO THEME
    Explicit toggle for dark mode, light mode, or follow system. Currently
    the app appears to always use a dark theme.

39. GRID ZOOM SLIDER
    A slider to continuously resize grid thumbnails from tiny (8+ columns)
    to large (1-2 columns) like Lightroom's thumbnail size control.
    Currently fixed at minimum 180pt width.

40. CUSTOMIZABLE SIDEBAR SECTIONS
    Reorder or hide sidebar sections (Library, Sources, Tags). Drag to
    reorder. Each section collapsible with state persisted.

41. PHOTO SET BADGES CUSTOMIZATION
    Let users choose which overlays appear on grid cells: show/hide format
    label, show/hide rating badge, show/hide flag badge, show/hide tag pills.

42. FULL-SCREEN CULLING MODE
    A dedicated full-screen mode that hides the sidebar and only shows
    the large image viewer with a slim toolbar. Maximizes screen real estate
    for culling sessions.

43. MENU BAR STATUS ITEM
    A menu bar icon showing current scan status, selected count, and quick
    actions (Add Source, Start Transfer). Lets you trigger operations
    without switching to the main window.

44. MULTIPLE WINDOWS
    Support multiple simultaneous DuckSort windows, each with an independent
    source and state. Useful for working on two different shoots side by side.

45. TOUCH BAR SUPPORT
    For MacBook Pros with Touch Bar: show rating stars, flag buttons, and
    tag hotkeys on the Touch Bar during culling.

46. KEYBOARD SHORTCUT CHEAT SHEET OVERLAY
    Press "?" to show a full-screen semi-transparent overlay listing all
    current keyboard shortcuts, including custom tag hotkeys.

47. PHOTO SET NOTES
    A freeform text note field per photo set (distinct from caption/XMP).
    Stored locally in the tag store JSON. Show in inspector. Searchable.

================================================================================
  PERFORMANCE & POWER FEATURES
================================================================================

48. BACKGROUND SCAN CONTINUATION
    Allow scanning to continue in the background while the user works.
    Show a progress indicator in the sidebar. Don't block the UI.

49. INCREMENTAL RESCAN
    On rescan, only re-process folders whose modification date has changed
    since the last scan. Skip unchanged directories entirely. Dramatically
    faster re-scans for large libraries.

50. PHOTO SET SORTING OPTIONS
    Sort the grid by: filename (current), capture date (EXIF), file size,
    rating, flag status, date added. Ascending or descending. Persist the
    chosen sort order.

51. STATISTICS DASHBOARD
    A panel showing shoot summary stats: total photos, total file size,
    breakdown by format (RAW/JPEG/HEIF), edited vs. unedited percentage,
    average rating, most-used tags.

52. BATCH METADATA EDIT
    Select multiple photo sets and set rating, flag, or tags for all of
    them at once. E.g., "Set all selected to 3 stars."

53. TRANSFER QUEUE
    Queue multiple transfer operations and run them sequentially. Start
    a move operation, then queue a second one — both run automatically
    without waiting for user input between them.

54. NETWORK SHARE SUPPORT
    Explicitly support scanning and transferring to/from SMB/NFS network
    shares (currently may work but isn't optimized for network I/O latency).

55. CLOUD STORAGE AWARENESS
    Detect when source or destination is a cloud-synced folder (iCloud Drive,
    Dropbox). Warn before moving files that might cause sync conflicts. Show
    sync status badge on source items.

================================================================================
  INTEGRATION & ECOSYSTEM
================================================================================

56. LIGHTROOM CATALOG AWARENESS
    Detect if any source folder is referenced in a Lightroom catalog (.lrcat)
    and warn before moving files, since it would break Lightroom's references.

57. APPLE PHOTOS INTEGRATION
    Option to import selected photo sets directly into the Apple Photos
    library after culling, using PHPhotoLibrary. Skip the export step for
    casual users.

58. SHORTCUTS APP SUPPORT
    Expose DuckSort actions as Apple Shortcuts actions: "Scan Folder",
    "Apply Tag", "Export Selected", "Get Photo Count". Enables automation.

59. DRAGGING OUT TO FINDER
    Allow dragging individual photo sets from the grid directly to Finder,
    other apps, or Mail as a drag-and-drop source. Currently only accepts
    drops, doesn't act as a drag source.

60. APPLESCRIPT / AUTOMATION SUPPORT
    Make the app scriptable with a basic AppleScript dictionary: get/set
    source folder, get selected photos, trigger export. For power users
    who automate their workflow.

================================================================================
  ACCESSIBILITY & USABILITY
================================================================================

61. VOICEOVER SUPPORT
    Full VoiceOver accessibility labels on all controls — photo set cells,
    tag buttons, rating controls, keyboard shortcuts. Make the app usable
    for visually impaired photographers.

62. ONBOARDING FLOW
    First-launch walkthrough: "Choose your shoot folder → Set up tags →
    Choose destination → You're ready!" Reduces friction for new users.

63. CONTEXTUAL TIPS
    Show helpful tips in the status bar or tooltip popovers when the user
    hovers over key UI areas (e.g., "Double-click to open full viewer",
    "Drag folders here to add as source").

64. UNDO FOR FILE MOVES
    After a move operation, show a timed "Undo" toast notification (like
    macOS's new Undo Move in Finder). Clicking it reverses the file moves.
